# jackett-sync-arr

## 特性

* 同步Jackett所有索引器到Radarr，Sonarr
* 支持对不同类型的索引器（public/private)设置不同的配置

## 开始

-   ```shell
      git clone https://github.com/CyberRim/jackett-sync-arr.git
    ```

-   ```shell
      cd jackett-sync-arr
    ```

-   ```shell
      npm install
    ```

-   修改配置文件 config/default.json

-   ```shell
      npm start
    ```
